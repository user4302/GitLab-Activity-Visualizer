# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-02

### Added
- Initial release of the GitLab Contribution Calendar Generator.
- Interactive landing page with live preview.
- Server-side SVG generation API (`/api/calendar`).
- Support for 4 color themes: Classic, Dark, Sky Blue, and Warm Orange.
- Month and Day labels on the generated SVG.
- One-click copy for Markdown embed codes.
- Responsive design for mobile and desktop.
- Netlify/Vercel compatibility.
